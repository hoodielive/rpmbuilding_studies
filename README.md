# Building RPM(s)

I need to be better at building rpms. I'm going to use this repo as an endpoint to push my studies and exercises. Yayyy!

### Required packages for Centos 8 and Red Hat 8:
```bash
dnf install gcc \
		rpm-build \
		rpm-devel \
		rpmlint \
		make \
		python \
		bash \
		coreutils \
		diffutils \
		patch \
		rpmdevtools
```

		rpm-devel \

		make \

		bash \
		coreutils \
